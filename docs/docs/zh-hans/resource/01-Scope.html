SDK适用范围

  • 设备需支持 linux 或 Android 系统
  • 支持TCP/IP协议栈

编译说明

  • linux平台：涂鸦根据客户toolchain编译成的相关的库，更新到github上。
  • Android平台：涂鸦根据客户提供的 API 版本号，使用NDK产生工具链，并编译成的相关的库，更新到github上。
